# Scope

## Project Scope

- The project will be conducted strictly in a controlled and authorized environment.
- Only test users or consent-based participants will be included in the simulation.
- The phishing simulation will involve email-based attack scenarios with a simulated landing page.
- Data collection will focus only on awareness metrics (opens, clicks, interactions) and not sensitive personal data.
- Infrastructure setup includes domain configuration, email delivery setup, and landing page hosting.
- Monitoring and analysis of user interaction behavior will be performed.
- Detection and incident response processes will be simulated and documented.
- The project excludes any real-world unauthorized phishing activities.
