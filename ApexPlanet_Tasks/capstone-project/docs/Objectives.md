# Objectives

## Project Objectives

- Simulate a controlled phishing campaign within an ethical and authorized environment.
- Evaluate user awareness and susceptibility to phishing techniques.
- Demonstrate the phishing attack lifecycle for educational and cybersecurity training purposes.
- Collect and analyze interaction metrics such as email opens, link clicks, and form submissions.
- Identify common user behavior patterns during phishing attempts.
- Provide actionable mitigation strategies to improve security awareness.
- Document the entire process including planning, execution, monitoring, and reporting as part of a professional cybersecurity capstone project.
